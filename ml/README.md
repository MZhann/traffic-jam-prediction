# trafficjam-ml

FastAPI service for traffic predictions.

## Quickstart (PowerShell)

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

## Quickstart (bash)

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

Health check: <http://localhost:8000/health>

OpenAPI docs (auto-generated by FastAPI): <http://localhost:8000/docs>
